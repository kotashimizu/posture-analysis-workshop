# Posture Analysis Workshop

PT・OT向け「コードを書かずに姿勢分析アプリを作る」ハンズオンワークショップの教材です。

90分のセミナー後半（60分のワークショップ部分）で、参加者全員が同じ完成形（MediaPipe使用の姿勢分析Webアプリ）に到達します。

## このワークショップで作るもの

- 患者さんの全身写真または動画から姿勢を客観評価するWebアプリ
- MediaPipeで33点のランドマークを自動検出
- Kendallの姿勢評価をベースに、頭部前方位・骨盤傾斜・膝のアライメント等を数値化
- 矢状面・前額面の視点を自動判別
- カメラでその場撮影してリアルタイム骨格表示
- すべてブラウザ内で処理、画像は外部送信しない

## 進め方

1. このフォルダを **AntiGravity** で開く
2. `prompts/00_最初に読む.md` を読む
3. `prompts/01_*.md` から順番に、中身をコピーしてAntiGravityに貼っていく
4. 各ステップで AntiGravity がコードを生成し、アプリが少しずつ完成していく
5. `prompts/09_*.md` まで終われば完成

詳しい手順は [prompts/00_最初に読む.md](./prompts/00_最初に読む.md) を読んでください。

## 必要なもの

- AntiGravity（GoogleのAI駆動IDE）
- Node.js（v20以上推奨）
- ブラウザ（Chrome / Edge / Safari）

詳細は [docs/02_必要ツール.md](./docs/02_必要ツール.md) を参照。

## 注意事項

- このアプリは触診ベースの姿勢評価の代替ではなく、客観指標を可視化する参考ツールです
- 最終的な姿勢評価および介入判断は、必ず資格を持つ専門職が行ってください
- 詳細は完成後アプリ内のディスクレーマー、または [docs/01_使い方.md](./docs/01_使い方.md) を参照
