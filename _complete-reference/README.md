# 姿勢分析Webアプリ

MediaPipe Pose Landmarkerで人物の姿勢ランドマークを検出し、Kendall姿勢評価の観察軸を参考に、角度と左右差を画面表示するWebアプリです。理学療法士が撮影条件を確認しながら、姿勢の傾向を客観的に整理する用途を想定しています。

## 使い方

1. 入力モードで「画像」または「動画」を選択します。
2. 視点選択で「矢状面（左向き）」「矢状面（右向き）」「前額面（前から）」「前額面（後ろから）」を選びます。
3. 撮影ガイドを確認し、ファイルをアップロードするかカメラで撮影します。
4. 動画の場合はスクラバーで時刻を選び、「このフレームで評価」を押します。
5. Canvas上の骨格と基準線、レポートの指標カード、自動生成所見を確認します。

## 撮影のコツ

- 被写体から3m離れ、全身が画面に収まるようにします。
- カメラは腰の高さに置き、水平を保ちます。
- 体のラインが見える服装にし、背景は無地にします。
- 矢状面は完全な側面、前額面は正対で撮影します。
- 回旋、遮蔽、暗い環境ではランドマーク精度が低下します。

## ディスクレーマー

本アプリの分析結果は、参考情報として提供されるものであり、医学的診断・評価の代替ではありません。
ランドマーク検出は2D画像からの推定であり、触診ベースの臨床評価とは精度が異なります。
最終的な姿勢評価および介入判断は、必ず資格を持つ専門職の判断に基づいて行ってください。
本アプリで処理される画像・動画はすべてブラウザ内で処理され、外部サーバーには送信されません。

## 開発手順

```bash
npm install
npm run dev
```

ブラウザで表示されたローカルURLを開きます。型確認と本番ビルドは次のコマンドで実行します。

```bash
npx tsc --noEmit
npm run build
```

## Netlifyデプロイ

1. GitHubにこのリポジトリをpushします。
2. Netlifyで対象リポジトリを選択します。
3. Build commandは `npm run build`、Publish directoryは `dist` を指定します。
4. `netlify.toml` のSPAリダイレクト設定により、任意のパスから `index.html` に戻ります。
