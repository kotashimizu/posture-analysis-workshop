# 02 必要ツール

ワークショップ参加に必要なものと、その入れ方。

---

## 必須

### 1. AntiGravity

GoogleのAI駆動IDE（統合開発環境）。
このワークショップではAntiGravityがコードを書きます。

- インストール：[AntiGravity公式サイト](https://antigravity.google.com/)（最新URLは事前案内をご確認ください）
- 起動して、Googleアカウントでサインイン
- 動作確認：適当なフォルダを開いて、チャットに「こんにちは」と打って返答が来るか

### 2. Node.js

JavaScriptを動かす土台。npm（パッケージマネージャ）が含まれます。

- インストール：[Node.js公式サイト](https://nodejs.org/)
- バージョン：v20以上 推奨
- 確認：ターミナルで `node -v` と打って `v20.x.x` のような表示

### 3. ブラウザ

完成したアプリをブラウザで動かします。

- 推奨：Google Chrome または Microsoft Edge
- Safari は一部機能で制限あり
- 最新版にしておく

---

## 推奨（あると便利）

### Git

バージョン管理ツール。失敗しても元に戻せます。

- Macには標準で入っている（`git --version` で確認）
- Windows：[Git for Windows](https://git-scm.com/download/win)

### ターミナル

コマンドを打つ場所。AntiGravity内蔵のターミナルでもOK。

- Mac：標準のTerminal、または iTerm2
- Windows：PowerShell（標準）、または Windows Terminal

---

## 不要（このワークショップでは使わない）

- Claude Code
- Codex
- VSCode（AntiGravityが代替）

---

## 事前確認

ワークショップ前に、以下が動くことを確認してください：

```bash
# Node.js が入っているか
node -v
# v20.x.x のような表示

# npm が入っているか
npm -v
# 10.x.x のような表示

# Git が入っているか（推奨）
git --version
# git version 2.x.x のような表示
```

すべてエラーなく表示されればOK。
