# 03 Kendall姿勢評価とは

PT・OTには馴染みのある内容ですが、ワークショップでアプリが何を計算しているかを理解するための解説です。

---

## Kendall姿勢評価の基本

Florence Kendall らが体系化した姿勢評価。
理想姿勢からの逸脱を、矢状面（横）・前額面（前後）から観察する。

### 矢状面の理想ライン（重力線）

被写体を横から見たとき、以下の点が一直線上に並ぶ：

- 耳垂（外耳道の中央）
- 肩峰
- 大転子
- 膝関節（少し前方）
- 外果（やや前方）

このラインからの逸脱を観察する。

### 主な姿勢タイプ

1. **理想アライメント**：すべての指標が許容範囲内
2. **後弯前弯型（Kyphosis-Lordosis）**：胸椎後弯増加 + 腰椎前弯増加 + 骨盤前傾 + 頭部前方位
3. **フラットバック（Flat-Back）**：腰椎前弯減少 + 骨盤後傾
4. **スウェイバック（Sway-Back）**：骨盤前方変位 + 上部胸椎後弯 + 腰椎前弯減少
5. **前弯強調型（Lordosis）**：腰椎前弯増加 + 骨盤前傾

### 前額面の評価

正面または背面から見たとき、左右対称性を観察：

- 頭部側方傾斜
- 肩高さ左右差
- 骨盤高さ左右差
- 膝のアライメント（内反 / 外反）
- 足部のアライメント

## このアプリでの実装

### MediaPipe 33ランドマークから取れるポイント

| ランドマーク | 番号 | Kendall評価での役割 |
|---|---|---|
| 鼻 | 0 | 頭部位置の参考 |
| 耳 | 7, 8 | 頭部前方位の起点（耳垂の代用） |
| 肩 | 11, 12 | 肩峰の代用 |
| 股関節 | 23, 24 | 大転子の代用 |
| 膝 | 25, 26 | 膝関節中心 |
| 足首 | 27, 28 | 外果の代用 |
| 踵 | 29, 30 | 足部評価 |
| 足先 | 31, 32 | 足部評価 |

### 触診ベースとの違い

- ASIS / PSIS の正確な触診点は取れない
- 服を着た体表のキーポイント
- 体型・服装で精度が変わる

このアプリは **触診の代替ではなく、客観指標の参考** という位置づけ。

## 計算している指標

### 矢状面

| 指標 | 計算 |
|---|---|
| 頭部前方位 | 耳→肩 ベクトルの鉛直からの角度 |
| 肩前方位 | 肩→大転子 ベクトルの鉛直からの角度 |
| 体幹前後傾 | 肩中点→腰中点 の鉛直からの角度 |
| 骨盤傾斜傾向 | 肩→腰→膝 の角度から推定 |
| 膝伸展度 | 腰→膝→足首 の角度（180° を中立として） |
| 足関節 | 膝→足首→足先 の角度（90° を中立として） |

### 前額面

| 指標 | 計算 |
|---|---|
| 頭部側方傾斜 | 左右耳のy座標差 |
| 肩高さ左右差 | 左右肩のy座標差 ÷ 体高 |
| 骨盤高さ左右差 | 左右腰のy座標差 ÷ 体高 |
| 膝アライメント | 腰-膝-足首 の内反/外反角度 |
| 足部 | 左右の踵-足先 の角度 |

---

## 仮閾値について

各指標には「正常 / 軽度逸脱 / 明らかな逸脱」の境界となる閾値があります。

このアプリの初期値は **仮の値**です。`src/lib/kendall/thresholds.ts` で調整できます。

### 調整の目安

- 高齢者対象：早期発見のため厳しめに
- スポーツ選手：より厳格な基準
- 産後女性：骨盤関連を厳しめに

施設・対象者に合わせて、臨床経験 or 文献値で調整してください。
