# 00 最初に読む

ワークショップを始める前に、このファイルを最後まで読んでください。

---

## このワークショップで作るもの

患者さんの全身写真または動画から、姿勢を客観評価するWebアプリです。

- MediaPipeで33点のランドマークを自動検出
- Kendallの姿勢評価をベースに、頭部前方位や骨盤傾斜などを数値化
- 矢状面・前額面の視点を自動判別
- カメラでその場撮影してリアルタイム骨格表示
- すべてブラウザ内で処理（画像は外部送信されない）

90分のワークショップで、参加者全員が同じ完成形に到達します。

---

## 進め方の全体像

```
このフォルダをAntiGravityで開く
   ↓

完成
```

## 事前チェック（最初に1分だけ）

Step 01に入る前に、Node.js がパソコンに入っているか確認してください。

ターミナル（AntiGravityのターミナルでもOK）で：

```bash
node -v
```

- `v20.x.x` のような表示が出たらOK、次に進んでください
- `command not found: node` と出たら、Node.js が入っていません。下の手順で入れてください

### Node.js が入っていない場合

**公式サイトからLTS版をインストール（一番簡単）**

1. https://nodejs.org/ を開く
2. 左側の「LTS」と書かれた緑のボタンからインストーラをダウンロード
3. ダウンロードしたファイルを開いて画面の指示に従う
4. 完了したらターミナルを再起動して `node -v` を再確認

**AntiGravityに頼む方法もあります**

AntiGravityに以下を貼ると、Homebrew経由のインストール手順を案内してくれます：

```
ターミナルで `node -v` と打ったら command not found: node と出ました。
Mac/WindowsにNode.js v20以上を入れたいです。手順を教えてください。
```

ただし時間がかかるため、**公式サイトからのインストールが推奨です**。

## 最初にやること

### 1. このフォルダをAntiGravityで開く

AntiGravityを起動して、`File` > `Open Folder` で `posture-analysis-workshop` フォルダ全体を開いてください。

### 2. ターミナルでこのフォルダに移動

AntiGravityのターミナル（または通常のターミナル）で：

**Mac の場合：**
```bash
cd ~/projects/posture-analysis-workshop
```

**Windows の場合：**
```powershell
cd $HOME\projects\posture-analysis-workshop
```

場所が違う場合は、自分が置いた場所に合わせてください。

### 3. AntiGravityのチャットを開く

画面右側か下側にチャット欄があります。ここにプロンプトを貼っていきます。

---

## プロンプトの貼り方

各 `prompts/0X_*.md` の中身は、おおむねこのような形式です：

```markdown
# 0X タイトル

## このステップの目的
〜

## AntiGravityに渡すプロンプト

（ここからコピーする部分）
〜〜〜
（ここまで）

## 完了したら確認すること
〜
```

「AntiGravityに渡すプロンプト」のブロックの中身を**そのまま**コピーして、AntiGravityのチャットに貼り付けて送信してください。

AntiGravityが「了解しました、〜を実装します」のような返答をして、ファイルを生成・編集していきます。
