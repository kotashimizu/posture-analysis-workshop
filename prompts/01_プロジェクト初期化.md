## 1. プロジェクト初期化（package.json、Vite設定など）

ルート直下に以下を作成してください：

- package.json
  - name: "posture-analysis-app"
  - private: true, type: "module", version: "0.1.0"
  - scripts: dev / build / preview / typecheck
  - dependencies: react ^19、react-dom ^19、lucide-react ^0.468.0
  - devDependencies: typescript ^5.7、vite ^6.0、@vitejs/plugin-react ^4.3、tailwindcss ^4.0、@tailwindcss/vite ^4.0、@types/react ^19、@types/react-dom ^19、@types/node ^22

- vite.config.ts
  - plugins: react()、tailwindcss()
  - resolve.alias: "@" を "./src" にマップ

- tsconfig.json と tsconfig.node.json
  - strict: true
  - jsx: "react-jsx"
  - moduleResolution: "Bundler"
  - paths: { "@/*": ["./src/*"] }
  - target: ES2020、lib: ES2020/DOM/DOM.Iterable

- index.html
  - 言語: ja
  - title: "姿勢分析アプリ"
  - viewport設定
  - main.tsx を読み込む

- .gitignore（既にあるので確認のみ）

## 2. Tailwind v4 + 自前shadcn風UIコンポーネント

`src/styles/globals.css` を作成し、以下を含める：

- @import "tailwindcss"
- :root の color-scheme: light、フォント設定（Inter、Noto Sans JP、Yu Gothic、Hiragino Sans、system-ui）、背景 #f8fafc
- box-sizing border-box
- body の word-break: keep-all、overflow-wrap: break-word、line-break: strict、background #f8fafc
- h1〜h4、button、.button、.label、[data-nowrap] には white-space: nowrap、word-break: keep-all、overflow-wrap: normal を適用
- canvas、video、img の max-width: 100%
- input[type="range"] の accent-color: #1e3a8a

`src/lib/utils.ts` を作成し、`cn()` ユーティリティ関数（クラス名を結合する関数、tailwind-mergeなどの依存なしでシンプルなフィルタ実装でOK）を定義してください。

`src/components/ui/` 配下に、Radix UIなどの外部依存を一切使わない自前のshadcn風コンポーネントを作成してください：

- Button（variant: primary / secondary / outline / ghost / danger、size: sm / md / lg）
- Card / CardHeader / CardTitle / CardDescription / CardContent / CardFooter
- Tabs / TabsList / TabsTrigger / TabsContent（ステート管理は React Context + useState）
- RadioGroup / RadioGroupItem
- Slider（HTMLの input[type="range"] をラップ）
- Badge（variant: default / success / warning / danger / outline、whitespace-nowrap必須）
- Alert（variant: default / warning / danger）
- Tooltip（hover表示）
- Dialog（modal表示、actionLabel/onAction を受け取る）
- Progress（HTMLの progress 風の見た目を div で）
- Label
- Separator

医療系の落ち着いたトーン（白背景、ダークブルー #1e3a8a / #0f172a、グレー #475569、薄グレー #e2e8f0、緑 #059669、橙 #ea580c、赤 #dc2626）を使ってください。

## 3. 型定義

`src/types/index.ts` を作成：

```ts
export type Severity = "normal" | "mild" | "severe";

export type ViewType =
  | "sagittal-left"
  | "sagittal-right"
  | "frontal-front"
  | "frontal-back";

export type MediaMode = "image" | "video";

export type Landmark = {
  x: number;
  y: number;
  z?: number;
  visibility?: number;
};

export type MetricDirection =
  | "neutral"
  | "forward" | "backward"
  | "anterior" | "posterior"
  | "flexion" | "hyperextension"
  | "left" | "right"
  | "medial" | "lateral";

export type EvalMetric = {
  id: string;
  label: string;
  value: number;
  displayValue: string;
  unit: "deg" | "percent" | "score";
  severity: Severity;
  summary: string;
  direction: MetricDirection;
  landmarkIndices: number[];
};

export type PostureType =
  | "ideal"
  | "kyphosis-lordosis"
  | "flat-back"
  | "sway-back"
  | "lordosis"
  | "unclassified";

export type EvalResult = {
  view: ViewType;
  postureType: PostureType | null;
  metrics: EvalMetric[];
  findings: string[];
};

export type DetectionState = "idle" | "loading" | "ready" | "error";

export type MediaSource =
  | { kind: "image"; url: string; element: HTMLImageElement }
  | { kind: "canvas"; url: string; element: HTMLCanvasElement };

export type VideoSource = { url: string; fileName: string };

## 4. ヘッダー・フッター・ディスクレーマー

`src/components/Header.tsx`：
- 左：ShieldCheckアイコン（lucide-react）+ アプリ名「姿勢分析アプリ」
- 右：注意喚起テキスト「参考ツールです。診断・治療の代替ではありません」
- 白背景、薄いボーダー

`src/components/Footer.tsx`：
- 中央：免責事項テキスト
- 「本アプリの分析結果は参考情報です。最終判断は専門職が行ってください。」
- 控えめな表示

`src/content/disclaimer.ts`：
- ディスクレーマー全文を文字列でエクスポート
- 内容：「本アプリの分析結果は、参考情報として提供されるものであり、医学的診断・評価の代替ではありません。ランドマーク検出は2D画像からの推定であり、触診ベースの臨床評価とは精度が異なります。最終的な姿勢評価および介入判断は、必ず資格を持つ専門職の判断に基づいて行ってください。本アプリで処理される画像・動画はすべてブラウザ内で処理され、外部サーバーには送信されません。」

`src/components/Disclaimer.tsx`：
- disclaimer.ts の文字列を表示するシンプルなコンポーネント
