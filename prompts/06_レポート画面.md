# 1. 姿勢タイプバッジ

`src/components/PostureTypeBadge.tsx` を作成：

```tsx
import { Badge } from "@/components/ui/badge";
import type { PostureType } from "@/types";

type Props = {
  postureType: PostureType | null;
};

const labels: Record<PostureType, string> = {
  ideal: "理想アライメントに近い傾向",
  "kyphosis-lordosis": "後弯前弯型に近い傾向",
  "flat-back": "フラットバック型に近い傾向",
  "sway-back": "スウェイバック型に近い傾向",
  lordosis: "前弯強調型に近い傾向",
  unclassified: "複合パターン",
};

export function PostureTypeBadge({ postureType }: Props) {
  if (!postureType) {
    return <Badge variant="outline">前額面評価</Badge>;
  }
  const variant = postureType === "ideal" ? "success" : "warning";
  return <Badge variant={variant}>{labels[postureType]}</Badge>;
}
```

# 2. 指標カード

`src/components/MetricCard.tsx` を作成：

- Props: metric: EvalMetric
- 構造（縦並び）：
  1. 上段：色ドット（severityで色変更）+ 指標名（data-nowrap）
  2. 中段：数値（text-3xl font-semibold tabular-nums data-nowrap）+ severity ラベル小（軽度/明らか/仮閾値内）
  3. 下段：所見テキスト（text-sm leading-6 text-slate-600）
- severityの色：
  - normal：bg-emerald-500（ドット）/ text-emerald-700（文字）
  - mild：bg-orange-500 / text-orange-700
  - severe：bg-red-500 / text-red-700
- severityラベル：
  - normal：「仮閾値内」
  - mild：「軽度」
  - severe：「明らか」

# 3. 所見テキスト表示

`src/components/FindingsText.tsx` を作成：

- Props: findings: string[]
- 「所見」見出し + 改行付きの文章ブロック
- 落ち着いた slate 色

# 4. レポートパネル

`src/components/ReportPanel.tsx` を作成：

```tsx
import { BarChart3 } from "lucide-react";
import { FindingsText } from "@/components/FindingsText";
import { MetricCard } from "@/components/MetricCard";
import { PostureTypeBadge } from "@/components/PostureTypeBadge";
import { Card, CardContent, CardDescription, CardHeader, CardTitle } from "@/components/ui/card";
import type { EvalResult } from "@/types";

type Props = { result: EvalResult | null };

export function ReportPanel({ result }: Props) {
  return (
    <Card>
      <CardHeader>
        <div className="flex flex-col gap-3 sm:flex-row sm:items-start sm:justify-between">
          <div>
            <CardTitle className="flex items-center gap-2">
              <BarChart3 className="h-5 w-5 text-blue-900" aria-hidden="true" />
              レポート
            </CardTitle>
            <CardDescription>
              Kendall姿勢評価を参考に、角度と左右差を仮閾値で整理します。
            </CardDescription>
          </div>
          {result ? <PostureTypeBadge postureType={result.postureType} /> : null}
        </div>
      </CardHeader>
      <CardContent>
        {result ? (
          <div className="space-y-5">
            <div className="grid grid-cols-1 gap-3">
              {result.metrics.map((metric) => (
                <MetricCard key={metric.id} metric={metric} />
              ))}
            </div>
            <FindingsText findings={result.findings} />
          </div>
        ) : (
          <div className="rounded-xl border border-dashed border-slate-300 bg-slate-50 p-6 text-sm leading-7 text-slate-600">
            検出が完了すると、姿勢タイプ、指標カード、自動生成所見がここに表示されます。
          </div>
        )}
      </CardContent>
    </Card>
  );
}
```

# 5. レイアウトを2カラムに

`src/App.tsx` で、PoseCanvas と ReportPanel を横並びにする：

```tsx
<div className="grid gap-6 xl:grid-cols-[minmax(0,1fr)_480px]">
  <PoseCanvas
    source={source}
    landmarks={landmarks}
    loading={pose.state === "loading"}
  />
  <ReportPanel result={result} />
</div>
```

xl 以上では左にプレビュー、右にレポート。それ未満では縦並び。

「レポート予定エリア」のCardは削除して、上の構造に置き換えてください。
