## ブラウザでの動作確認チェックリスト

`npm run dev` で起動した状態で、以下を1つずつ確認してください。

### 基本動作

- [ ] ページが表示される（http://localhost:5173）
- [ ] 初回ロード時にディスクレーマーが表示される
- [ ] 「同意して始める」で閉じる、リロードしても再表示されない

### 入力UI

- [ ] 「画像」「動画」タブが切り替わる
- [ ] ファイルドロップゾーンが表示される
- [ ] 視点エリアに「判別前（画像を読み込むと自動判別します）」が表示

### 画像モード

- [ ] 全身が映った画像をドロップ → プレビューに画像+骨格が表示
- [ ] 視点が自動判別される（「自動判別: 矢状面（左向き）」など）
- [ ] レポートに姿勢タイプバッジが表示
- [ ] レポートに6個（矢状面）or 5個（前額面）の指標カードが表示
- [ ] 各カードに色ドット・数値・severity・所見
- [ ] 所見テキストが表示

### カメラ撮影

- [ ] 「カメラ起動」でカメラ映像（縦長）が表示
- [ ] **骨格がリアルタイムで描画される**
- [ ] 「シャッター」で撮影 → プレビューに骨格付き画像
- [ ] 動画モードで「録画開始」「録画停止」が動く

### 動画モード

- [ ] 動画ファイルをアップロード → スクラバーが表示
- [ ] スライダーで時刻指定 → 「このフレームで評価」で評価される
- [ ] 「ベストフレーム自動選択」で進捗表示 → 自動シーク → 評価

### エラーハンドリング

- [ ] 全身が入っていない画像だと「ランドマークを検出できません」のエラー
- [ ] カメラ拒否すると「カメラを起動できません」のエラー

## 完成おめでとう

ここまでお疲れさまでした。

このアプリでできること：

- MediaPipeで姿勢ランドマークを自動検出
- Kendall姿勢評価をベースに各指標を数値化
- 矢状面・前額面の自動判別
- カメラでリアルタイム骨格表示
- 動画から最も静止しているフレームを自動選択
- すべてブラウザ内で処理（画像は外部送信なし）

今日のセミナーで触ったこのアプリは、皆さんの「出発点」です。
