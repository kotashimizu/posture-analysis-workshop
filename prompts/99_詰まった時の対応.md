# 99 詰まった時の対応

ワークショップ中に詰まったら、まずこのファイルを見てください。

---

## まずは深呼吸

エラーが出ても焦らないでください。AI開発では「エラーを見て、聞き直す」が日常です。
落ち着いて以下のテンプレを試してください。

---

## 1. AntiGravityがエラーを返してきた時

エラーメッセージをそのままコピーしてAntiGravityに貼ってください。

```
さっきの実装でエラーが出ています。直してください。

{エラーメッセージをそのままコピペ}
```

---

## 2. ブラウザの画面が真っ白

ブラウザで `F12`（または右クリック → 検証）を押してコンソールを開き、赤いエラーメッセージをコピーしてください。

```
ブラウザで開いたら画面が真っ白で、コンソールに以下のエラーが出ています。
原因と直し方を教えてください。

{エラーメッセージをコピペ}
```

---

## 3. 思った動きと違う

```
「{こうしたい}」と思っていたのですが、実際は「{こうなっている}」です。
原因を教えて、直してください。
```

---

## 4. ファイルが見つからない / 何が変わったか分からない

```
さっきの実装で、何のファイルがどう変わったか教えてください。
それぞれのファイルが何の役割か、初心者向けに簡単に説明してください。
```

---

## 5. 全部リセットしたい

```
さっきの変更をすべて取り消して、一つ前の状態に戻してください。
```

または、ターミナルで以下を実行：

```bash
git status     # 何が変わったか確認
git restore .  # 全部元に戻す
```

---

## 6. プロンプトをもう一度貼ったらどうなる？

同じプロンプトを2回貼ると、AntiGravityは「もう実装済みです」と教えてくれることが多いです。
中途半端に止まった場合は、もう一度貼っても問題ありません。

---

## 7. 専門用語が分からない

```
「{用語}」とは何ですか。このアプリでどう使われているかも、PT・OT初心者向けに教えてください。
```

---

## 8. AntiGravityが止まった / 反応がない

- 1分くらい待ってみる（裏で処理しています）
- それでも反応がなければ、画面右上のリロードボタンでチャットを再起動

---

## 9. それでも分からない

ZoomチャットでHELPと書いてください。
講師（kota）が画面共有でフォローします。

---

## 万能の一言

詰まったら、これを試してみてください：

```
よく分からないので、何が起きていて、何をすればいいか、順番に教えてください。
```

これだけでAntiGravityは状況を整理して、次の一手を提案してくれます。
