# 撮影サンプル素材

ワークショップで使えるサンプル素材です。
自分で撮影した素材がない時、または動作確認用にお使いください。

## 同梱ファイル

| ファイル | 用途 |
|---|---|
| `矢状面.png` | 矢状面（横向き）の静止画サンプル |
| `矢状面予備.png` | 矢状面の予備画像 |
| `前額面.png` | 前額面（正面）の静止画サンプル |
| `前額面予備.png` | 前額面の予備画像 |
| `前屈動画.mp4` | 動画モード・ベストフレーム検索の動作確認用 |

## 使い方

1. アプリの「画像」または「動画」タブで、ファイルアップロード欄にドラッグ&ドロップ
2. または「ファイルを選択」でこのフォルダ内のファイルを指定

## 注意

- これらの素材は本ワークショップの動作確認用です
- 配布・転載・SNS掲載はご遠慮ください
- 自院や自身の素材で動作確認する方が、現場に近い感覚を得られます
